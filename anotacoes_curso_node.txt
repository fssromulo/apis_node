Curso de Node.Js -- Criação de Api's
========================================


== Express oque é? == 

É um microframework, que vai auxiliar a trabalhar com rotas e views. Rotas são os endereços que o usuário poderá acessar através das api's. E views: São as formas de visuazilação.

// Primeiro parametro do método GET é a rota que será usada
// Segundo é uma função que receberá dois parametros "req" e "res"
// - REQ: Simboliza a requisicao que estavamos fazendo ao servidor, o req vai conter todos os detalhes dessa requisição, como por exemplo: Parâmetros, corpo da requisição, cabeçalho da requisição, usuário da requisição, autenticação se esta autenticado ou não, ip.
// - RES: Estarão todos os dados de respostas da requisição para o usuário.

app.get('/', (req, res) => {});


// Nodemon - "npm -D nodemon"
É Usada para não precisar ficar rodando "node server.js"
para ele funcionar devemos criar dentro do package.json um script como exemplo: "dev" e adicionar o comando 'nodemon server.js' que chamará nosso arquivo JS da api. Após isso basta rodar no cmd 'npm run -dev';
